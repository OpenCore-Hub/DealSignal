# SSO 单点登录

## Description
支持 SAML / OIDC 单点登录，满足企业客户对工作区成员统一身份管理的需求。

## Source
P2: SSO

## Hard Constraints
- 无针对本 Issue 的额外硬约束；遵循 PRD 全局安全与隐私约束。

## Acceptance Criteria
- [ ] 支持 SAML 2.0 与 OIDC 身份提供商
- [ ] 管理员可配置 SSO 元数据
- [ ] SSO 用户首次登录自动加入工作区
- [ ] 支持强制 SSO 登录

## Validation
- [ ] 通过 SSO 登录后成功进入工作区
